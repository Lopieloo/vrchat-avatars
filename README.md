# Lopieloo — VRChat Avatar Commissions & 3D Character Art

Hi, I'm **Lopieloo** — I make **custom VRChat avatars**: original 3D character
models with hand-crafted custom animations, rigged and optimized for both **PC and Quest**.

This repo is my portfolio and model archive — finished art, character renders, and
commission info in one place.

## 🎨 What I make
- **Original 3D characters** built from your concept or reference art
- **Full rigging** — Unity SDK3 / Avatars 3.0, expression menus, PhysBones
- **Custom animations** — emotes, idles, dances, toggles
- **PC + Quest optimization** so your avatar runs smooth everywhere
- Delivered as a **ready-to-upload Unity package**

## 🛠️ Commission process
`concept → model → rig → animate → deliver`
